Important FAQ

What is the Document Opject Model (DOM)?

The DOM is a tree structure of web documents, enabling JavaScript to dynamically modify content, structure, and styles.

How do you select an element from the dom

getElementById select id getElementsByClassName select class name, querySelector select one item and querySelectorAll select all item

What is even delegation in the contex of dom and why its usefull

Event delegation is a technique of attaching an event listener to only one, usually parent item. An event is happened on a child element, routers up to parent and then the parent handle the event.

How do you manipulate an element's attribute and style using the DOM?

the DOM methods such as getAttribute(), setAttribute(), removeAttribute(), and manipulating the style property, you can effectively change an element's attributes and styles dynamically. This is a fundamental part of creating interactive web pages.
